# 盘口异动与主力资金跟随策略

## 核心理念

盘口异动数据反映了**主力资金在关键价位的真实意图**。本技能通过东方财富盘口异动接口，实时获取全市场/个股的异动信号，量化个股买卖盘强度，形成可执行的跟随策略。

## 异动类型分类

所有异动类型按**买卖方向**和**强度**分为三类：

## 强度评分体系

### 个股盘口强度分

```
强度分 = 买入信号加权和 - 卖出信号加权和
```

其中每个信号的权重 = 基础强度分 × 时间衰减因子

### 时间衰减因子

| 时间区间 | 衰减系数 |
|----------|----------|
| 09:30~11:00 | 1.0（上午有效信号） |
| 11:00~13:30 | 0.8（午间） |
| 13:30~14:30 | 1.0（下午盘） |
| 14:30~15:00 | 1.2（尾盘加分，主力尾盘行为更真实） |

### 强度分档

| 强度分 | 判定 | 建议 |
|--------|------|------|
| ≥ 20 | 极强 | 主力明确做多，可跟随买入 |
| 10 ~ 19 | 偏强 | 主力试探做多，可列入观察 |
| -9 ~ 9 | 中性 | 多空平衡，观望 |
| -19 ~ -10 | 偏弱 | 主力试探离场，减仓 |
| ≤ -20 | 极弱 | 主力明确做空，清仓回避 |

### 快照回放评分方法

快照回放是**盘中实时评分**的标准流程：每来一条新异动信号，就以当前时刻为锚点，对过去30分钟窗口做一次完整评分，判断是否满足买入条件。

#### 核心规则

- **窗口大小**：固定30分钟滑动窗口（以新信号时间为右边界）
- **评分锚点**：每笔新信号触发一次评分，而非定时轮询
- **向后看**：只使用窗口中已有的信号，不使用未来数据

#### 评分步骤（每笔新信号到达时执行）

```
步骤1：划定窗口
  窗口 = [当前时间 - 30分钟, 当前时间] 内的所有异动信号

步骤2：计算买卖强度
  买入强度 = Σ(买入信号基础分 × 时间衰减)
  卖出强度 = Σ(卖出信号基础分 × 时间衰减)
  净强度   = 买入强度 - 卖出强度

步骤3：统计买入类型
  核心买入类型 = 火箭发射 / 大笔买入 / 封涨停板 / 有大买盘 / 快速反弹
  （排除：竞价上涨/下跌、高开/低开5日线、缺口类、60日新高/低等非订单驱动型信号）
  买入类型数 = 窗口中出现的核心买入类型种数

步骤4：计算异动均价与偏离度
  异动均价 = 窗口内所有核心买入信号的成交价格均值
  偏离度  = (当前价 - 异动均价) / 异动均价 × 100%

步骤5：检查买入条件（全部满足才触发）
  ① 净强度 ≥ 15
  ② 核心买入类型数 ≥ 2
  ③ |偏离度| ≤ 2%
  ④ 窗口内无高台跳水/大笔卖出等严重卖出干扰（定性辅助）
```

#### 评分示例模板

```
快照 @10:27:46 火箭发射 21.85 (+2.49%)
  窗口 [09:57 ~ 10:27] 共3笔信号
  买入: 快速反弹×2(12分) + 火箭发射×1(10分) = 22分
  卖出: 无 = 0分
  净强度: 22 - 0 = 16
  买入类型: 快速反弹, 火箭发射 → 2种 ✅
  异动均价: (21.38 + 21.37 + 21.85) / 3 = 21.53
  偏离度: (21.85 - 21.53) / 21.53 = +1.5% ✅
  结论: 条件①✅ ②✅ ③✅ → 开仓信号触发
```

#### 注意事项

- 同一股票在一天内可能多次达标/失效，每次以新信号到达时的窗口状态为准
- 30分钟窗口滚动会导致旧信号移出、买入类型数变化，之前达标后续可能失效
- 尾盘14:50后不再开新仓（风控规则优先）
- 卖出信号（高台跳水/大笔卖出）不单独构成否决条件，但会压低净强度、叠加判断

## 主力资金跟随策略

### 买入触发条件（需同时满足）

1. 个股盘口强度分 ≥ 15
2. 最近30分钟内有≥2次买入类异动（不含竞价型信号）
3. 当前价格未远离异动均价 ±2%
4. 当日大盘无系统性风险（非千股跌停/股灾状态）
5. 个股非ST、非新股上市前5日

### 卖出触发条件（触发任一即可）

1. 持仓股实时强度分 ≤ -15
2. 出现"高台跳水"或"大笔卖出"且强度分转负
3. 持仓60分钟内出现连续3笔卖出信号
4. 盘中分时跌破异动密集区均价 -3%

### 风控规则

- 单笔亏损达 -5% 无条件止损
- 当日连续触发2次止损信号，停止交易至次日
- 尾盘14:50后不出新仓
- 单股持仓不超过总资金 20%

## 买入策略（多信号交叉验证）

`scripts/buy_signal.py` 是对 `大笔买入` 单一条件策略的优化替代。核心理念：**不看单一大单，看多种信号的合力**。

### 与旧方案的核心差异

```python
# 旧方案：只看一种信号 + 固定金额
df = aks.stock_changes_em(symbol="大笔买入")
if 累计额 > 8e7: 触发

# 新方案：多信号加权 + 买卖对比
买入信号: 大笔买入(8) + 火箭发射(10) + 有大买盘(7) + ...
卖出信号: 高台跳水(10) + 大笔卖出(8) + 有大卖盘(7) + ...
净强度 = Σ买入加权 - Σ卖出加权
触发条件: 净强度≥20 且 ≥2种买入类型 且 买入强度>卖出强度×1.5
```

### 信号强度对照

| 类型 | 旧方案 | 新方案 |
|:---|:---:|:---:|
| 大笔买入 | 核心信号 | 8分 |
| 火箭发射 | 忽略 | 10分（最优先） |
| 有大买盘 | 忽略 | 7分 |
| 高台跳水 | 忽略 | 卖出10分（一票否决级） |
| 有大卖盘 | 忽略 | 卖出7分（减分项） |
| 龙虎榜 | 硬过滤器 | 可选加分 |

### 使用

```bash
# 全市场扫描买入候选
python3 scripts/buy_signal.py

# 严格模式：净强度≥50 + ≥3种买入类型才触发
python3 scripts/buy_signal.py --min-net 50 --min-types 3

# 查看详细评分
python3 scripts/buy_signal.py --json
```

详细设计参考 `references/buy_strategy_optimization.md`。

## 注意事项

- 盘口异动数据为**实时盘中**数据，盘后可能不可用
- 异动信号以东方财富推送为准，存在一定延迟（通常1~3秒）
- 主力资金可能制造假信号（如小单托底、大单出货），需结合价格区间分析
- 本技能提供分析参考，不构成投资建议
